DQN (Deep Q-Network) Algorithm Demo

DQN이란?

Deep Q-Network (DQN)은 강화학습의 한 방법으로, 딥러닝과 Q-Learning을 결합한 알고리즘입니다. 에이전트가 환경과 상호작용하면서 보상을 최대화하는 행동을 학습합니다.

주요 특징:

  • Experience Replay: 과거 경험을 저장하고 무작위로 샘플링하여 학습
  • Target Network: 안정적인 학습을 위한 별도의 타겟 네트워크 사용
  • ε-greedy 정책: 탐험과 활용의 균형을 위한 행동 선택 전략

What is DQN?

Deep Q-Network (DQN) is a reinforcement learning method that combines deep learning with Q-Learning. The agent learns to take actions that maximize rewards through interaction with the environment.

Key Features:

  • Experience Replay: Stores past experiences and samples them randomly for learning.
  • Target Network: Uses a separate target network for stable learning.
  • ε-greedy Policy: An action selection strategy to balance exploration and exploitation.

DQN 알고리즘 단계

  1. 환경 관찰: 현재 상태 s를 관찰합니다
  2. 행동 선택: ε-greedy 정책으로 행동 a를 선택합니다
  3. 행동 실행: 환경에서 행동을 실행하여 보상 r과 다음 상태 s'를 얻습니다
  4. 경험 저장: (s, a, r, s') 튜플을 경험 버퍼에 저장합니다
  5. 배치 학습: 경험 버퍼에서 배치를 샘플링하여 Q-네트워크를 업데이트합니다
  6. 타겟 업데이트: 주기적으로 타겟 네트워크를 업데이트합니다

DQN Algorithm Steps

  1. Observe Environment: Observe the current state s.
  2. Choose Action: Choose action a using the ε-greedy policy.
  3. Execute Action: Execute the action in the environment to get reward r and next state s'.
  4. Store Experience: Store the (s, a, r, s') tuple in the experience buffer.
  5. Batch Learning: Sample a batch from the experience buffer to update the Q-network.
  6. Update Target: Periodically update the target network.
