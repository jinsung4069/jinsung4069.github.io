머신러닝 알고리즘 시뮬레이션

다양한 머신러닝 알고리즘이 데이터를 어떻게 학습하고 분류하는지 직접 체험해보세요.

Machine Learning Simulations

Interactive demonstrations of various machine learning algorithms.

Learning Categories

지도학습과 비지도 학습으로 나누어 체험하기 Explore Supervised and Unsupervised Learning

정답 라벨을 바탕으로 예측 규칙을 배우는 지도학습과, 라벨 없이 데이터의 구조와 취향 패턴을 발견하는 비지도 학습을 구분해 살펴볼 수 있습니다.

Compare supervised learning, which learns from labeled answers, with unsupervised learning, which discovers structure and preference patterns without labels.

Supervised

지도학습

  • 선형 회귀: 연속값 예측
  • KNN, 의사결정트리, 나이브 베이즈: 분류 경계 체험
  • 라벨이 있는 데이터로 예측 규칙 학습
Unsupervised

비지도 학습

  • 군집화: K-Means, 밀도 기반, 계층적 클러스터링
  • 추천 시스템: 협업, 콘텐츠, 모델, 딥러닝 기반 필터링
  • 시각화와 차원축소: PCA와 커널 기반 투영
Supervised Learning

지도학습 체험 Supervised Learning Demos

정답 라벨이 주어진 데이터를 이용해 값과 클래스를 예측하는 과정을 직접 조작해봅니다.

Use labeled data to see how models predict values and classes.

선형 회귀 (Linear Regression)

데이터 포인트들의 경향성을 가장 잘 설명하는 하나의 직선을 찾습니다. 캔버스를 클릭하여 점을 추가해보세요. 실시간으로 최적의 회귀선이 계산됩니다.

Finds the best-fitting straight line through the points. Click on the canvas to add points. The regression line updates in real-time.

기울기(Slope): 0
절편(Intercept): 0

K-최근접 이웃 (KNN)

새로운 데이터의 클래스를 결정할 때, 가장 가까운 K개의 이웃 데이터를 참조합니다. 버튼으로 클래스를 선택하고 점을 찍어보세요. 마우스를 움직이면 해당 위치의 분류 결과를 볼 수 있습니다.

Classifies new data based on the majority class of its K nearest neighbors. Select a class, add points, and move your mouse to see the classification.

의사결정트리 (Decision Tree)

데이터의 속성(여기서는 X, Y 좌표)을 기준으로 영역을 분할하여 분류합니다. 점을 찍으면 자동으로 공간을 직사각형 영역으로 분할하여 분류 경계를 만듭니다.

Splits the data space into rectangular regions to classify points. Add points to see how the tree partitions the space.

나이브 베이즈 (Naive Bayes)

각 클래스의 데이터가 가우시안(정규) 분포를 따른다고 가정하고 확률을 계산합니다. 점을 찍으면 배경색으로 각 위치에서의 확률적 분류 결과를 보여줍니다.

Calculates probabilities assuming data follows a Gaussian distribution. Add points to visualize the probabilistic classification boundaries.

Unsupervised Learning

비지도 학습 체험 Unsupervised Learning Demos

라벨 없이 데이터의 묶임, 비슷한 취향, 숨은 요인, 낮은 차원 표현을 찾아내는 과정을 살펴봅니다.

Explore clustering, recommendations, latent factors, and lower-dimensional structure without predefined labels.

K-평균 군집화 (K-Means)

데이터를 K개의 클러스터로 묶는 알고리즘입니다. 점을 찍고 '실행'을 눌러보세요. 중심점(X)이 이동하며 군집을 형성하는 과정을 볼 수 있습니다.

Partitions data into K distinct clusters. Add points and click 'Run' to watch the centroids (X) move and form clusters.

밀도 기반 클러스터링 (DBSCAN)

반경 안에 충분한 이웃이 모인 곳을 하나의 군집으로 확장합니다. 흩어진 점은 노이즈로 남습니다.

Expands clusters from dense neighborhoods and leaves isolated points as noise.

클러스터: 0
노이즈: 0

계층적 클러스터링 (Hierarchical Clustering)

가장 가까운 군집부터 차례대로 합치며, 거리 임계값에 따라 몇 개의 군집으로 남는지 확인합니다.

Repeatedly merges the closest groups and shows how the distance threshold changes the cluster count.

클러스터: 0
마지막 병합 거리: -

사용자-콘텐츠 평점 행렬

평점 칸을 클릭하면 1점부터 5점, 빈칸 순서로 바뀝니다.

협업 필터링 기반 추천

협업 필터링 및 추천 시스템

나와 비슷한 사용자가 높게 평가한 콘텐츠를 추천합니다. 행렬의 빈칸은 예측해야 할 선호도입니다.

Recommends items that similar users liked. Blank cells are preferences to estimate.

비슷한 사용자: -

콘텐츠 특징과 사용자 프로필

콘텐츠 기반 추천

콘텐츠 기반 필터링

콘텐츠의 특징 벡터와 사용자의 선호 벡터를 비교해 가장 잘 맞는 항목을 추천합니다.

Compares content feature vectors with a user preference vector to rank items.

모델 기반 협력 필터링

평점 행렬을 사용자 요인과 콘텐츠 요인으로 분해해 숨은 취향 축을 학습합니다.

Factorizes the rating matrix into user and item factors to learn hidden taste dimensions.

반복: 0
추천: -

최근 관심 콘텐츠

딥러닝 기반 추천 점수

딥러닝 기반 필터링

클릭한 콘텐츠 이력을 임베딩으로 압축하고 비선형 은닉층을 거쳐 다음 추천 점수를 계산합니다.

Compresses clicked item history into embeddings and uses a nonlinear hidden layer to score recommendations.

은닉층 활성화: -

시각화와 차원축소 (PCA, Kernel)

3차원 특징 데이터를 2차원 화면으로 투영합니다. PCA는 분산이 큰 축을 찾고, 커널 투영은 비선형 구조를 드러냅니다.

Projects 3D feature data into 2D. PCA finds high-variance axes, while kernel projection exposes nonlinear structure.

설명 분산: -
투영 방식: PCA
