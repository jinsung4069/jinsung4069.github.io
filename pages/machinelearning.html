머신러닝 알고리즘 시뮬레이션

다양한 머신러닝 알고리즘이 데이터를 어떻게 학습하고 분류하는지 직접 체험해보세요.

Machine Learning Simulations

Interactive demonstrations of various machine learning algorithms.

선형 회귀 (Linear Regression)

데이터 포인트들의 경향성을 가장 잘 설명하는 하나의 직선을 찾습니다. 캔버스를 클릭하여 점을 추가해보세요. 실시간으로 최적의 회귀선이 계산됩니다.

Finds the best-fitting straight line through the points. Click on the canvas to add points. The regression line updates in real-time.

기울기(Slope): 0
절편(Intercept): 0

K-최근접 이웃 (KNN)

새로운 데이터의 클래스를 결정할 때, 가장 가까운 K개의 이웃 데이터를 참조합니다. 버튼으로 클래스를 선택하고 점을 찍어보세요. 마우스를 움직이면 해당 위치의 분류 결과를 볼 수 있습니다.

Classifies new data based on the majority class of its K nearest neighbors. Select a class, add points, and move your mouse to see the classification.

K-평균 군집화 (K-Means)

데이터를 K개의 클러스터로 묶는 알고리즘입니다. 점을 찍고 '실행'을 눌러보세요. 중심점(X)이 이동하며 군집을 형성하는 과정을 볼 수 있습니다.

Partitions data into K distinct clusters. Add points and click 'Run' to watch the centroids (X) move and form clusters.

의사결정트리 (Decision Tree)

데이터의 속성(여기서는 X, Y 좌표)을 기준으로 영역을 분할하여 분류합니다. 점을 찍으면 자동으로 공간을 직사각형 영역으로 분할하여 분류 경계를 만듭니다.

Splits the data space into rectangular regions to classify points. Add points to see how the tree partitions the space.

나이브 베이즈 (Naive Bayes)

각 클래스의 데이터가 가우시안(정규) 분포를 따른다고 가정하고 확률을 계산합니다. 점을 찍으면 배경색으로 각 위치에서의 확률적 분류 결과를 보여줍니다.

Calculates probabilities assuming data follows a Gaussian distribution. Add points to visualize the probabilistic classification boundaries.
